# Croatian/Balkan Gender Names Dataset

A manually curated dataset of **Croatian first names mapped to gender**.  
This project helps you enrich and analyze your customer or user databases by automatically assigning gender based on first names.

Ideal for:

- Customer analytics & reporting  
- Demographic segmentation  
- CRM data enrichment  
- E-commerce personalization  
- Marketing insights  
- Data cleanup & standardization
